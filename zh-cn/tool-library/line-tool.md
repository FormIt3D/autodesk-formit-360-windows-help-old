# 直线

使用线分割面，并构造几何图形。

键盘：L

![](../.gitbook/assets/line_toolbar.png)

可以在 X、Y 或 Z 轴上绘制线，也可以在现有几何图形上绘制线。捕捉到轴时，一条彩色线将延伸到正在绘制的线之外，以便指示正在哪个轴上绘制。可以在绘制时看到线的长度，并且可以通过按 **Tab** 键并输入值来手动输入特定长度。

按轴定义线颜色：

要指定长度，只需键入尺寸值，然后“编辑尺寸”对话框将显示。单击“确定”或按 Enter 键接受该值。

* X = 红色
* Y = 绿色
* Z = 蓝色
