# 矩形

可以使用**“创建草图”**工具栏菜单中的**“矩形”**工具，快速绘制矩形面。拉伸面以创建矩形体积。

键盘快捷键：R

使用轴可确保在正确方向上绘制：

可以在 X、Y 或 Z 轴上绘制矩形。绘制时，只需键入值（这将显示“编辑尺寸”对话框），即可指定尺寸。点击 Enter 键或按“确定”以接受尺寸。

要绘制矩形，请单击以在草图中设置第一个点。

按轴定义线颜色：

* X = 红色
* Y = 绿色
* Z = 蓝色
