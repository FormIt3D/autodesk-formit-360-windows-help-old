# 矩形

您可以使用**「建立草圖」**工具列功能表中的**矩形**工具快速繪製矩形面。擠出面以建立矩形體積。

鍵盤快速鍵：R

使用軸可確保您以正確的方向繪製：

您可以在 X、Y 或 Z 軸上繪製矩形。繪製時，只需鍵入值，即可指定尺寸，這會顯示「編輯尺寸」對話方塊。按 Enter 或按「確定」接受尺寸。

若要繪製矩形，請按一下以設定草圖中的第一點。

依軸區分的線顏色：

* X = 紅色
* Y = 綠色
* Z = 藍色
