# 陣列

您可以在 FormIt 中建立線性和徑向陣列。若要**讓邊、面或整個物件形成陣列**，請從關聯式功能表中按一下「陣列」圖示。接下來，選擇「線性」或「徑向」，指定其他參數，然後按一下物件以開始產生陣列。再按一下以放置陣列元素。
