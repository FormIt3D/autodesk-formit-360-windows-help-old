# 測量

**測量**工具位於[工具列](../formit-introduction/tool-bars.md)上。

您可以使用**測量**工具來測量模型中的既有幾何圖形。您也可以使用**測量**工具對幾何圖形指定尺寸。

1. 選取**「測量」**工具。
2. 選取兩點以測量它們之間的距離。
3. 選取要調整的面或邊，按一下顯示的值，然後在顯示的對話方塊中加入新值。
