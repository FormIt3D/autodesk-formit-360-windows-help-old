# 第 I 部分

## Mies van der Rohe 的 Farnsworth 住宅建模

在本 Primer 的第 I 部分中，我们将介绍基本的 FormIt 工具和工作流，以在不到两个小时的时间内构建一座历史悠久的现代主义风格住宅！有关此处所讨论的工具、工具栏、选项板和流程的详细说明，请参见**“FormIt 简介”**和**“工具库”**部分。
