# 개념 솔리드 모델링

설계 옵션을 통해 작업할 개념 형상을 쉽고 빠르게 작성할 수 있습니다.

FormIt의 솔리드 모델링 커널은 놀라운 시간 절약 워크플로우를 지원하여 모델링을 쉽고 효율적으로 수행할 수 있도록 해줍니다.

**솔리드 모델링이 유용한 이유와 Revit과 같은 다운스트림 응용프로그램에서 사용할 수 있도록 모델이 솔리드인지 확인하는 방법을 알아보십시오.**
